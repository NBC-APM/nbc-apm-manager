⚡ ZEP 미니게임 진행 가이드

❓ OX 퀴즈 (ZEP 미니게임)

1
사전 문제 준비

튜터/매니저 TMI 기반 문제를 엑셀 파일로 작성 후 ZEP OX 퀴즈 앱에 업로드

2
앱 실행 & 세팅

ZEP 스페이스 내 OX 퀴즈 맵에서 미니게임 앱 실행 → 업로드한 문제 파일 선택

3
게임 진행

진행자가 문제 읽어주며 진행 / 정답 공개 후 탈락자 확인 / 1등은 앱 내 기록 확인 가능

4
등수 기록

1등 외 등수는 앱에서 확인 불가 → 진행 중 별도 채팅/메모로 순위 기록 필요

⚠️ 1등 외 순위는 앱에서 자동 기록 안 됨 — 진행 중 수동 기록 필수

🎨 그림 빙고 (자체 진행)

1
빙고판 사전 제작

5×5 빙고판에 25개 키워드 배치 → 팀별 1장씩 사전 배포 (PPT 또는 이미지 파일)

2
키워드 발표

진행자가 키워드 1개씩 발표 → 각 팀이 해당 칸을 찾아 그림으로 표현 후 체크

3
빙고 달성 확인

3빙고 달성 팀이 "빙고!" 외침 → 화면 공유 or 사진 전송으로 빙고판 검증

4
우승팀 발표

진행자 최종 확인 후 우승팀 발표 및 상품 지급

⚠️ 키워드당 제한시간(약 45초) 타이머 화면 공유 권장

📻 라디오 세션

1
사연 사전 수집

구글폼으로 성함 / 익명·실명 선택 / 사연 제목·내용 / 노래 신청 접수

2
사연 선정

수집된 사연 중 7개 내외 선정 → 라디오 진행 PPT에 사연 내용 추가

3
투표 앱 세팅

ZEP 투표 앱에서 사연 투표 항목 사전 세팅 (제목: 가장 인상깊은 사연 투표)

4
라디오 진행 & 투표

PPT 보며 사연 낭독 → 전체 투표 진행 → 1등 사연 발표 및 상품 지급

⚠️ 사연 보낸 분께 포인트 500 지급 여부 사전 결정 필요

🔤 ZEP 퀴즈 게임 (택1)

1
게임 선택

초성 퀴즈: 가볍고 빠른 진행 / 객관식 퀴즈: 정답 명확, 안정적 / 퀴즈 골든벨: 다양한 형식 가능

2
문제 준비

선택한 게임에 맞게 문제 사전 준비 → ZEP 미니게임 앱에 업로드 or 실시간 입력

3
앱 실행

ZEP 스페이스 내 해당 미니게임 앱 실행 → 참여자 전원 입장 확인 후 시작

4
결과 확인

게임 종료 후 순위 확인 → 상위 입상자 포인트 지급

⚠️ 3가지 중 1개만 선택 — 회의에서 최종 결정 필요
