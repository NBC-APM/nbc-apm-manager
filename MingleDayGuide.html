사연 라디오 진행 가이드

🎙️ 밍글데이 라디오

발표회를 마친 수강생들의 솔직한 이야기를 라디오 형식으로 공유하는 코너입니다.
진행자가 DJ처럼 사연을 읽고, 공감과 위로의 분위기를 만들어주세요.

😭 고생했어요 사연
발표 준비하면서 힘들었던 순간, 밤샘 작업 이야기
😂 웃긴 에피소드
발표 중 실수담, 팀원과의 웃긴 해프닝
💌 감사 전하기
튜터/매니저/팀원에게 전하고 싶은 감사 메시지
🌟 나에게 한마디
수료 후 자신에게 전하고 싶은 응원 메시지
🤔 솔직 고백
부트캠프 하면서 가장 후회한 것 / 잘한 것
🎵 신청곡
지금 이 순간 듣고 싶은 노래 + 사연 한 줄
1
사연 사전 수집 (이벤트 3~5일 전)
구글 폼 또는 노션 폼으로 사연 받기. 카테고리별로 분류하고, 읽어도 되는지 동의 여부 확인. 익명 처리 옵션도 제공하기.
2
사연 큐레이션 (이벤트 1~2일 전)
수집된 사연 중 분위기에 맞는 것 선별. 웃긴 것 → 감동적인 것 순서로 배치. 너무 개인적이거나 민감한 내용은 제외.
3
라디오 진행 멘트 준비
각 사연 앞뒤로 DJ 멘트 준비. 예: "다음 사연은 익명의 OO 수강생분이 보내주셨어요~" / 사연 읽은 후 공감 코멘트 한 줄씩 준비.
4
실시간 사연도 받기 (선택)
젭 채팅창에 실시간으로 사연 받는 코너도 추가 가능. 진행자가 채팅 모니터링하며 즉석에서 읽어주는 방식. 매니저 1명이 채팅 모니터링 담당.
🎵 라디오 분위기 연출 팁
  • 배경음악은 잔잔한 lo-fi 또는 감성 팝으로 낮게 깔기
  • 진행자 목소리 톤을 평소보다 부드럽고 천천히 유지
  • 사연 읽기 전 "잠깐 조용히 들어봐요~" 같은 멘트로 집중 유도
  • 감동 사연 후에는 잠깐 침묵 + 이모지 리액션 유도 ("💜 눌러주세요~")
  • 신청곡은 실제로 틀어주면 더 좋음 (유튜브 공유 화면)
🎮
젭(ZEP) 미니게임 목록
🙋
OX 퀴즈
전체 참여 · 10~15분
젭 맵에 O/X 구역 설정. 진행자가 문제 읽으면 수강생이 이동해서 답 선택. 틀리면 탈락, 마지막 생존자 우승.

추천 문제 유형:
· 부트캠프 관련 상식 퀴즈
· 디자인 트렌드 OX
· "이 사람이 한 말이다 OX" (튜터/수강생 명언)
전체참여 젭 기본기능 쉬운난이도
⚡
스피드 퀴즈
팀전 · 15~20분
팀을 나눠 한 명이 설명, 나머지가 채팅으로 답 입력. 제한 시간 내 가장 많이 맞힌 팀 우승.

추천 주제:
· 디자인 툴 이름 맞히기
· UI 용어 설명하기
· 부트캠프 추억 키워드
팀전 채팅활용 중간난이도
🎯
이모지 빙고
개인전 · 15분
사전에 빙고판(3x3 또는 4x4) 배포. 진행자가 이모지/키워드 하나씩 부르면 해당하는 칸 체크. 빙고 완성되면 채팅에 "빙고!" 입력.

빙고 칸 예시:
· 밤샘한 적 있다 / 커피 3잔 이상 / 발표 떨렸다
개인전 사전준비필요 쉬운난이도
🔤
초성 퀴즈
전체 참여 · 10분
진행자가 초성 공개, 채팅에 먼저 정답 입력한 사람이 포인트 획득. 젭 채팅창 활용.

추천 주제:
· 디자인 관련 단어 (ㅍ ㄹ ㅌ ㅇ = 프로토타입)
· 부트캠프 관련 단어
· 유명 디자이너/브랜드
채팅활용 빠른진행 쉬운난이도
🗺️
젭 보물찾기
개인전 · 15~20분
젭 맵 곳곳에 오브젝트(힌트판) 숨겨두기. 수강생들이 맵을 돌아다니며 힌트 수집 후 최종 정답 채팅 입력.

힌트 예시:
· 각 구역마다 단어 조각 숨기기
· 조각 모으면 문장 완성
젭맵활용 사전세팅필요 탐험형
🎨
이 디자인 누구꺼?
전체 참여 · 10~15분
수강생들의 발표 결과물 일부를 화면 공유. 누구의 작품인지 채팅으로 맞히기. 디자인 부트캠프 특화 게임!

진행 팁:
· 사전에 수강생 동의 받기
· 작품 일부만 크롭해서 공개
· 작가 본인이 직접 설명하는 시간도 추가
부트캠프특화 화면공유 소통형
✨
추가 온라인 이벤트 아이디어
🗳️
실시간 투표 이벤트
Mentimeter / 슬리도 활용
멘티미터나 슬리도로 실시간 투표 진행. 결과를 화면 공유하며 반응 유도.

투표 주제 예시:
· 부트캠프 기간 중 가장 힘들었던 주차는?
· 나를 가장 잘 표현하는 이모지는?
· 다음에 또 하고 싶은 프로젝트 유형은?
외부툴활용 실시간반응
🏅
수강생 어워즈
사전 투표 + 발표
수강생들이 서로 투표해서 시상하는 코너. 진지한 상보다 웃긴 상 위주로!

추천 시상 카테고리:
· 가장 열정적이었던 수강생
· 가장 많이 웃겼던 수강생
· 밤샘 황제/황후
· 피드백 장인
· 갓생 살았던 수강생
사전준비필요 감동보장
📸
포토부스 이벤트
젭 포토존 활용
젭 맵에 포토존 구역 만들기. 수강생들이 아바타로 포즈 잡고 스크린샷 찍어 채팅에 공유.

포토존 아이디어:
· 졸업 포토존 (학사모 오브젝트)
· 팀별 단체 포토존
· 튜터/매니저와 함께 찍기
젭맵활용 추억만들기
🎤
1분 소감 릴레이
자발적 참여형
원하는 수강생이 마이크 켜고 1분 소감 발표. 강요 없이 자발적으로. 진행자가 분위기 띄워주기.

진행 팁:
· 튜터/매니저가 먼저 시범 보이기
· 소감 후 박수 이모지 유도
· 너무 길어지면 부드럽게 마무리 유도
자발참여 감동형
🎰
랜덤 경품 추첨
클로징 이벤트
이벤트 참여 포인트 합산 or 단순 랜덤 추첨. 화면 공유로 추첨 과정 공개.

추첨 툴:
· 구글 랜덤 번호 생성기
· 휠오브네임즈 (wheelofnames.com)
· 랜덤 추첨 사이트
클로징 경품이벤트
💬
익명 칭찬 릴레이
따뜻한 마무리 코너
사전에 구글 폼으로 특정 수강생에게 익명 칭찬 메시지 받기. 진행자가 낭독하면 해당 수강생 리액션.

진행 팁:
· 모든 수강생이 최소 1개씩 받도록 유도
· 클로징 직전에 배치하면 감동 마무리 가능
감동보장 사전준비필요

메인 진행자 (1명)
전체 흐름 진행, 사연 낭독, 게임 규칙 설명, 분위기 리드. 목소리 톤 관리 중요. 매니저 또는 진행 경험 있는 수강생 추천.

젭 채팅창 실시간 확인. 게임 정답 확인, 실시간 사연 수집, 문제 발생 시 진행자에게 전달. 매니저 담당 추천.
🖥
기술 담당 (1명)
화면 공유, BGM 조절, 젭 맵 관리, 투표 툴 세팅. 이벤트 중 기술 오류 대응. 사전에 리허설 필수.
📋
점수/기록 담당 (1명)
게임 점수 집계, 우승자 기록, 추첨 명단 관리. 스프레드시트 또는 노션 DB 활용.
